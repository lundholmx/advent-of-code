# Advent of Code

Code for AoC challenges.

## Requirements

- Python 3.10+
- Just (optional)

Install requirements into python virtual env:
```sh
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Running

Use the just recipy `run`:
```sh
# Run solution for day 12, year 2021
just run 21 1
...
```
